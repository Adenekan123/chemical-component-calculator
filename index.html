Chemical component calculator;

Instructions

  • Open terminal, Run: git clone https://github.com/Adenekan123/chemical-component-calculator.git
  • Run: cd chemical-component-calculator
  • Run: code . to open the project on VSCode
  • Open VSCode terminal
  • Run: npm install
  • Run: git branch branch name to create a new branch.
  • Run: git switch branch name to switch to the newly created branch
  • Run: npm start
  • Start coding the UI.

Note: Visit https://tailwindcss.com/ for more information on how use your styles
